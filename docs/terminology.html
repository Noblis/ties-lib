TIES Terminology

Top Level Information:
Data pertaining to the export such as the exporting system, schema version, time, and description

Object Items: Property Name: objectItems Media (video, images, documents, etc.) output from the exporting system

Authority Information: Property Name: authorityInformation Media (video, images, documents, etc.) output from the exporting system

Annotations: Property Name: annotations Information used to identify/organize object items and provide evaluative information

Supplementals Description: Property Name: supplementalDescriptions Other data associated with object items, such as motion detections, optical character recognition, etc.

Object Groups: Property Name: objectGroups Defines groups of related object items and object groups

Object Relationships: Property Name: objectRelationships Identifies the relationship present between two object items

Other Information: Property Name: otherInformation Key/value pairs of user-defined metadata
